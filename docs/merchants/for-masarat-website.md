# Masarat website team — link kit

Use this page when adding a public link from [masarat.ly](https://masarat.ly/) (or related Masarat sites) to the **MITF WooCommerce** merchant documentation.

## Canonical documentation URL

Set the link to your **published GitLab Pages** URL after the first successful deploy, for example:

`https://<namespace>.gitlab.io/<project_name>/`

GitLab shows the exact URL under **Deploy → Pages**. Keep **one** canonical HTTPS URL in the footer and integration pages to avoid duplicate SEO and outdated mirrors.

**GitHub repository (source of docs):** [github.com/anstwechy/public_online_payment_docs](https://github.com/anstwechy/public_online_payment_docs)

- “MITF WooCommerce payment plugin — merchant guide”  
- “Accept MITF card payments in WooCommerce (documentation)”
